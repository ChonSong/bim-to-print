## Luyten 3D (Australia — HQ in NSW)

| Field | Detail |
|-------|--------|
| **Type** | Gantry printer |
| **Build envelope** | Up to 100 m vertical reach; 1.5 m × 1.5 m build envelope |
| **Print speed** | Claims 60–70% faster than traditional formwork |
| **Price range** | $300,000–$500,000 AUD (estimated) |
| **Mobility** | Gantry system — requires site prep but re-deployable |
| **Material** | Standard concrete mix compatible |
| **Website** | <https://luyten3d.com> |
| **Best for** | Direct purchase — in-house printing capability |

### Key considerations

- Large vertical reach enables multi-storey components
- 1.5 m × 1.5 m envelope means segments must be joined
- Gantry requires level site and overhead clearance
- Training / onboarding package likely needed

## CyBe Construction (Netherlands — exporting to AU)

| Field | Detail |
|-------|--------|
| **Type** | Fixed robotic arm printer |
| **Build envelope** | Robot-arm reach (approx 2–3 m radius) |
| **Print speed** | Moderate — optimised for precision |
| **Price** | From €49,000 EUR (starter kit with mixing unit) |
| **Deployments** | Several European projects; being evaluated for AU pilots |
| **Website** | <https://cybe.eu> |
| **Best for** | Low-cost entry / R&D — fixed installation in workshop |

### Key considerations

- Cheapest entry point by far
- Fixed arm limits on-site deployment vs gantry
- Starter kit includes mixing unit (important for consistent concrete)
- Partner with local concrete supplier for material qualification
- Request demo unit shipped to east coast Australia

## Recommended approach for Sydney pilot

1. **Short-term (1–3 mo):** Partner with **Contour3D** — lowest risk, existing regulatory pathway, local Sydney operator
2. **Medium-term (6–12 mo):** Evaluate **CyBe** as in‑house R&D printer (€49k entry is negligible compared to build costs)
3. **Scale-up (12 mo+):** If in‑house capability proven, purchase **Luyten** or **PaCompatible** gantry for production
